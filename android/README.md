# Lexi — приложение для изучения английских слов

Android-приложение с карточками и интервальным повторением (SM-2, как в Anki).
Полностью офлайн: 1000 слов с определениями на английском (в стиле Cambridge)
встроены в приложение.

## Стек
- Kotlin + Jetpack Compose, Material 3 (динамические цвета, тёмная тема)
- Room (локальная БД), kotlinx.serialization (парсинг сидового JSON)
- Navigation Compose, ручной DI (без Hilt, чтобы держать сборку простой)

## Как открыть
1. Установите Android Studio (Ladybug/Koala или новее).
2. File → Open → выберите папку `android/`.
3. Android Studio предложит создать Gradle Wrapper — согласитесь (в проекте
   нет `gradlew`/`gradle-wrapper.jar`, так как их сборка требует обращения
   к `services.gradle.org`, недоступного в среде, где создавался проект).
4. Дождитесь Gradle Sync, запустите на эмуляторе или устройстве (minSdk 26 / Android 8.0+).

## Структура
```
app/src/main/java/com/lexi/vocab/
  data/local/        — Room: сущности, DAO, база данных
  data/repository/    — VocabRepository: сидирование, очередь повторений
  data/model/         — ReviewRating (Again/Hard/Good/Easy)
  srs/                — Sm2Scheduler — алгоритм интервального повторения
  ui/home/            — главный экран (сводка на сегодня)
  ui/review/          — экран повторения (свайп-карточки)
  ui/dictionary/      — поиск по словарю
  ui/stats/           — стрики и прогресс
  ui/theme/           — Material 3 тема
app/src/main/assets/dictionary_seed.json — офлайн-словарь (1000 слов)
```

## Откуда взят словарь
Как договаривались — не парсили Cambridge Dictionary напрямую (это
нарушало бы их условия использования). Вместо этого:

- **Список слов** — частотный список Google (`google-10000-english`,
  публичный домен), отфильтрован от служебных/грамматических слов.
- **Определения и примеры** — датасет
  [wordset-dictionary](https://github.com/wordset/wordset-dictionary),
  лицензия **CC BY-SA 4.0**, в основе — WordNet 3.0 (Princeton University).
- **IPA-транскрипция** — сгенерирована локально библиотекой `eng-to-ipa`
  (на основе словаря CMU), без обращения к интернету.

⚠️ CC BY-SA требует указания авторства. Перед публикацией в Google Play
стоит добавить экран «О приложении» с упоминанием Wordset/WordNet и
ссылкой на лицензию — я это не стал делать по умолчанию, чтобы не
навязывать конкретную формулировку, но структура экрана Stats — хорошее
место для такой ссылки.
